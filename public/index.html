Image Search Abstraction Layer

Search for images and see the past 10 search queries and when they were made.


Example Usage:

  • Pass the search query in the API URL along with an offset number to paginate through the responses like this:
    https://smm-image.glitch.me/api/searchimage/Black Panther?offset=1

  • In your JSON response you will receive 10 results per page with links for the image, thumbnail, context page, and a title snippet of the image.

  • Get a JSON response of the 10 most recently searched queries and the date and time of search like this: https://smm-image.glitch.me/api/latest/imagesearch
